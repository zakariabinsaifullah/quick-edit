== Description ==

Quick Edit is an awesome and super cool plugin which makes a "Quick Edit" dashboard widget and allows you to add any post or page to "Quick Edit" widget. You can easily edit your post or page quickly from your dashboard.

### Included Items

1. **Quick Edit Dashboard Widget** - A "Quick Edit" dashboard widget is generated from where you can easily edit your added post or page.
2. **Add Metabox** - A checkbox metabox is added in all posts and pages. So that you can easily add any page or post to "Quick Edit" dashboard widget for furthur quick edit.

### Features

This plugin includes the following features-

* Easily add any post or page to "Quick Edit" dashboard widget.
* You can easily and quickly edit added post or page from this dashboard widget.

== Installation ==

It's really easy and super simple to install **Quick Edit** plugin.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `Quick Edit`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download Quick Edit.
2.  Extract the `extra-widgets.zip` file. You'll get plugin files inside `Quick Edit` directory.
3.  Upload the plugin files to the `/wp-content/plugins/Quick Edit` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
